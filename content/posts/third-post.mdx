# GitHub Pages 部署指南

本文介绍如何将 Next.js 博客部署到 GitHub Pages。

## 步骤

1. 在 GitHub 创建仓库
2. 推送代码到 main 分支
3. 配置 GitHub Actions 自动部署

## 工作流

GitHub Actions 会自动：
- 安装依赖
- 构建项目
- 部署到 Pages

这样每次推送代码后，网站就会自动更新！
