# Tailwind CSS 使用心得

Tailwind CSS 是一个实用优先的 CSS 框架，让我来分享一些使用心得。

## 快速上手

只需要几个类名，就能创建美观的界面：

```html
<button class="bg-blue-500 hover:bg-blue-700 text-white font-bold py-2 px-4 rounded">
  点击我
</button>
```

## 响应式设计

使用前缀轻松实现响应式：

- `sm:` - 小屏幕
- `md:` - 中等屏幕
- `lg:` - 大屏幕

非常方便！
